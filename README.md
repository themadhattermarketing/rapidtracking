# Rapid Tracking

This is a minimal Flask application to manage sales prospects and commissions.

## Setup

```bash
pip install -r requirements.txt
python app.py
```

Open `http://localhost:5000` in your browser. The first login expects that a Rep with
name `admin` exists; you can create it directly in the database or adjust the code.

This project is a starting point and does not implement all requested features.
